# Security

## Unix Socket

- Socket at `/var/run/opd/opd.sock`, owned by `opd` group
- Only users in `opd` group can connect
- No passwords needed for local admin

## Agent Authentication

- Agent listens on `127.0.0.1:7070` only — never exposed externally
- All requests from daemon to agent are signed with HMAC-SHA256
- Header: `X-Agent-Token: HMAC-SHA256(body+timestamp, secret)`
- Replay protection: requests older than 30s are rejected

## File System

- Path traversal protection in agent: all paths resolved and checked against base dir
- Servers isolated in `/var/lib/opd/servers/{id}/`

## Audit Log

- Every action (start, stop, backup, console command) written to `events` table
- Includes: server_id, event_type, payload, timestamp

## Secrets

- `.env` never committed to repository
- `OPD_AGENT_SECRET` generated by `install.sh` (openssl rand -hex 32)
